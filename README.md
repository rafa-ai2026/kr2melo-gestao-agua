# KR²MELO Gestão de Água v5.1.6

## Impressão com cores fortes e alto contraste

Esta atualização mantém os dados salvos no navegador com a mesma chave `kr2melo.hidrometro.v1`.

### Melhorias aplicadas aos boletos impressos

- Texto e valores em preto reforçado, com peso maior para leitura nítida.
- Bordas, linhas de cobrança, assinatura e guias de corte em preto intenso.
- Cabeçalho com faixa vermelha da KR²MELO e preenchimento cinza forte.
- Campos de leitura e condomínio a pagar com preenchimento reforçado.
- Total com fundo preto e texto branco, mantendo contraste mesmo em impressoras que reduzem cores.
- Compatibilidade com impressão colorida e monocromática.

### Antes de imprimir

1. Se disponível na janela de impressão do navegador, ative **Gráficos de plano de fundo**.
2. Use **A4 · Paisagem · Escala 100%**.
3. Mantenha margens padrão ou mínimas; não use “Ajustar à página”.
4. Atualize o site com `Ctrl + F5` após substituir os arquivos.
